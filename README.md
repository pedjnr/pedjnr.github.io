Site pessoal de Pedro Carvalho, desenvolvido para reunir informações profissionais, trajetória, projetos, competências e formas de contato em um único endereço.

## Sobre

Este projeto é um portfólio pessoal construído como uma aplicação front-end com React e Vite. A interface utiliza componentes próprios, dados centralizados em arquivos JavaScript, estilos responsivos em CSS e suporte a tema claro/escuro.

## Tecnologias

- React
- Vite
- JavaScript
- CSS
- Lucide React
- Fontsource
- GitHub Pages

## Estrutura

```txt
src/
  components/   Componentes da interface
  data/         Conteúdo editável do site
  hooks/        Hooks auxiliares
  App.jsx       Composição principal
  index.css     Estilos globais

public/
  stack-icons/  Ícones usados na seção de competências
```
